Aqui são as três formas de colocar uma imagem dentro de uma <DIV>:

  • Background-color: skyblue;
  • Background-image: linear-gradiente(to bottom, yellow, orange);
  • Background-image: url("site-ou-local-que-você-tem-a-imagem.jpg");

A tag <background-image: url('')>, ela NÃO é semântica. Ou seja, ela não tem significado. Ela é uma FORMA para mostrar uma imagem.

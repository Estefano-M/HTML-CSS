Esse tipo de Style aqui é o mais flexível(o melhor também) para ser usado.

Para deixar assim eu fui na área de <head> e lá adicionei a tag <link:css> e coloquei a página .css que eu criei. E não importa quantas páginas o documento tenha, eu posso adicionar a mesma página CSS e todas irão ter as mesmas configurações de Styles.

Inclusive caso eu tenha outra configuração de CSS(outra página.css), eu posso adicionar mais uma tag <link:css> para que seja adicionado também essa configuração de style extra.

EVITE AO MÁXIMO USAR O STYLE INLINE(NA MESMA LINHA DAS TAGS). Só use se for fazer uma configuração bem pontual naquele arquivo e na mesma página.

Posso usar também a tag <style h1> nessa página principal dentro do <head> para usar uma configuração que só essa página aqui vai ter. Como por exemplo o Título Sublinhado(olhe lá em cima ↑) mas as outras páginas não serão afetadas.

Posso fazer outra configuração específica usando o inline quando por exemplo o dono da webpage quer que um parágrafo em específico esteja alinhado á direita como esse exemplo aqui em baixo Ir para página 2 . Veja no VS Code como usei a tag <style="text-align: right;"> dentro do parágrafo P.

OU SEJA, NO MESMO ARQUIVO(INDEX/PÁGINA) POSSO USAR O CSS EXTERNO, CSS INTERNO E CSS INLINE para fazer configurações específicas.

Usando Links Externos

Você pode acessar esse link para ouvir um som e ir estudar. Estou gostando de estudar ouvindo um song tranquilo de fundo. Usei aqui a tag <A href="url do site">.

O problema do link acima é que ele SUBSTITUI a sua página e a probabilidade de o usuário voltar para a sua página é pequena.

O que vou fazer aqui agora é gerar um novo link mas quando o usuário usá-lo, vai abrir uma nova aba e não vai perder sua webpage de vista. Segue video abaixo:

Eis aqui o exemplo.

NOW, LET ME BREAK IT TO YOU: Nesse novo link eu usei a mesma tag só que usei mais duas tags e ficou assim o link para um site externo: <A href="url do site" target="_blank" rel="external">.

O TARGET="_BLANK" é para abrir uma nova aba (o alvo é a nova aba).

o TARGET="" suporta os seguintes valores:

_BLANK e vai abrir o link em uma nova janela em branco.

_SELF e vai abrir um link na janela ou frame atual.

_TOP vai desfazer todos os frames e abrir o destino no navegador completo.

_PARENT similar ao uso do _top em uma referência á janela mãe.

NOME-DO-FRAME caso esteja usando frames, indicar o nome da janela a abrir.

O REL="EXTERNAL" é pra deixar claro para os SEO(Search Engine Optimization) que ele é um link para um site externo.

Outra coisa, na tag <A> a gente pode adicionar a tag hreflang="en" que permite indicar qual é o idioma principal do site para onde o link está desviando o fluxo de navegação. Isso permite avisar para os navegadores e softwares de tradução como lidar caso o usuário opte por traduzir automaticamente os conteúdos.

SEMPRE QUE FOR ABRIR UM SITE QUE NÃO É SEU, USAR TARGET significando uma nova aba E REL significando que é um site externo

IMPORTANTE: SEMPRE DEIXAR AS TAGS EM LETRAS MINÚSCULAS


Links Internos Aula 10 2/3

Está é a minha primeira página.

Vou criar um LINK INTERNO que vai redirecionar para uma outra página MINHA que está no mesmo servidor.

Para ir para a segunda página clique aqui e encontrar novos conteúdos.

Usei a tag <A href="index-da-pág-2" rel ="next"> o REL="NEXT" indica para o SEO que aquele link que o usuário vai clicar vai levar para a próxima página. E nas próximas páginas que irei criar e o usuário queira voltar, o outro index vai ter o REL="PREV" (de previous).

QUANDO EU FAÇO UM LINK INTERNO, EU NÃO PRECISO DA URL INTEIRA.

Quando estou navegando dentro do meu próprio site, na hora de utilizar a tag <a href="index-pag-3" target="_blank" rel="next">, eu posso mudar a tag TARGET para TARGET="_SELF" para indicar ao SEO que essa página é minha e não precisa abrir em uma nova guia. Mesmo que eu deixe somente TARGET="_BLANK" vai funcionar normalmente mas como nós queremos estar em um NÍVEL PROFISSIONAL, por isso temos que fazer assim.

Para conseguir criar esse link-download em PDF na primeira lista, eu usei a tag <a href="pasta-onde-esta-o-arquivo-para-download-selecionei-o-arquivo" download="coloquei-o-nome-do-arquivo-para-download" type="application/pdf">

A tag DOWNLOAD="arquivo-a-ser-baixado" é o arquivo a ser baixado que estava dentro do endereço(pasta) do A HREF.

A tag TYPE="application/pdf" é o formato do arquivo que você está criando um link-download.

Aqui nessa aula 15 01/04, falamos sobre os SELETORES PERSONALIZADOS.

Eles nos permite IDENTIFICAR UMA TAG por exemplo o h1 do título dessa página Criando Sites em HTML e CSS para caso eu queira usar mais de uma tag h1 mas que eu consiga realizar uma configuração somente nela COMO SE FOSSE O CSS INLINE mas como já sabemos, só para casos bem específicos como no exemplo de colocar a próxima página no lado direito da tela.

Para isso, eu sei a tag <ID> dentro da tag <h1> e depois fui no CSS e usei o h1#nome-do-id-que-voce-criou e assim, consegui fazer um style específico para ele. Veja como ficou: ↓

        

            /** Tudo em HTML que for "ID" e tudo em CSS é "#" */
            /** Tudo em HTML que for "class", em CSS é "."    */
            /** O nome dessa configuração que fizemos aqui em baixo se chama "Seletor Personalizado" */

            h1#titulo{
                text-align: center;
                background-color: rgb(61, 119, 61);
                color: white;
            }

Viu como ficou? Posso colocar qualquer nome dentro do ID para identifica-locomo já diz o próprio nome da tag.

Mas para aplicar esse ID no CSS, temos que usar o # como mostra o exemplo.

DENTRO DE UM MESMO DOCUMENTO HTML SÓ PODE TER UM ELEMENTO COM O MESMO ID. Por exemplo, não pode ter dois ID com o nome principal. Posso criar outro ID mas com outro nome, entendeu?

Posso usar somente em outra página(documento). FUNCIONA mas está ERRADO. A regra do W3C diz para somente usar 1 por documento caso a gente queira estar dentro do padrão de buscas do SEO.

FALANDO SOBRE A TAG CLASS

Caso eu queira usar um style no HTML básico e no CSS básico dessa página usando a mesma tag, eu posso usar a tag CLASS e assim conseguir configurar os dois h2.

Você pode classificar algo e não precisa usar a tag H1 ou outra tag no CSS, simplesmente coloque o .e-o-nome-da-classificação-que-você-fez e assim, todas as classificações que você fez, você consegue configura-las todas de uma vez. Veja exemplo: ↓

            
                /** Quando se trata da tag "CLASS", não precisa colocar o "h1,h2,h3"...ou outra tag alem do ponto e sua classificação como por exemplo aqui que o nome da classificação foi basico e SEMPRE USAR SEM ACENTUAÇÃO!*/
                /** Class também é um Seletor Personalizado*/

                .basico{
                    color: rgba(32, 80, 32, 0.973);
                    
                    
                }

                .intermediario{
                    color: rgb(221, 221, 45);
                    
                    
                }

                .avancado{
                    color: rgb(209, 73, 73);
                    
                }

Viu?!

Também consigo aplicar MAIS DE UMA CLASS NO MESMO ELEMENTO! É só separar ela com um espaço e aplicar o nome da nova class por exemplo:

            
                h1 class="avancado destaque">Falando sobre a tag h1>

Viu?! Tem duas classes dentro da tag H1.

Posso usar UM ID E UMA CLASS juntas mas o ID vai sobrepor a class.

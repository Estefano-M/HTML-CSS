- Pseudo-Classes - Aula 15 03/04

Aprendemos aqui sobre as PSEUDO-CLASSES. Pense nelas como se fossem REGRAS ESPECIAIS QUE SÓ ACONTECEM EM SITUAÇÕES ESPECÍFICAS.

            
                div:hover{
                    background-color: red;
                }

Vamos observar esse exemplo aqui de cima. Quando você PASSA O MOUSE EM CIMA, ele tem um comportamento. Eis o código dele: ↑

Exemplo: Imagine que você tem um botão e você quer que ele mude de cor SÓ QUANDO VOCÊ PASSAR O MOUSE POR CIMA. Olhe o código para entender ↓

            
                button: hover{
                    background-color: red;
                }

Isso quer dizer: Quando alguém passar o mouse por cima do botão, ele vai ficar vermelho.

PSEUDO-CLASSE = COMPORTAMENTO QUE É ATRIBUIDO EM SITUAÇÕES ESPECÍFICAS

Lista de Pseudo-Classe:

  • :hover
    • Quando o mouse está em cima do elemento
  • :active
    • Quando você está clicando no elemento
  • :focus
    • Quando um campo de texto está sendo usado
  • :first-child
    • Seleciona o primeiro filho(veja no style que vai ter um exemplo) de uma lista ou grupo
  • :last-child
    • Seleciona o último filho

<#> é um ID, <.> é uma classe e <:> é uma pseudo-class.

Posso usar a pseudo-class em outros elementos além do DIV como nesse exemplo no começo da página. Olha esse exemplo aqui: ↓.

Testando 1,2 e 3

Olha o que dá pra fazer também: ↓

            
                #d1:hover{
                    background-color: blue;
                }
                #d2:hover{
                    background-color: yellow;
                }

Aqui em cima, ao passar o mouse em cima do elemento <ID="d1">, vai ficar com a cor blue e no outro exemplo, com a cor amarela.

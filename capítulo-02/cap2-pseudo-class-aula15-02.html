Como pode perceber aqui nesse exemplo, fizemos que ao passar o mouse em cima do texto, um texto escondido fosse exibido.

Veja aqui como ficou o código ↓ ou veja no VS CODE.

        
            div:hover{
                color: red;
            }       

            div > p{
            display: none;
            color: white;
            background-color: red;
            width: 156px;
            }

            div:hover > p{
            display: block;
            
            }

Eis aqui um exemplo de CHILD, ou seja, é uma TAG DENTRO DA CLASS DIV e sua forma de representar é um um sinal de ">".

Quando quisermos que algo dentro da class div tenha um comportamento, iremos usar a forma child. E para saber mais, veja no VSCODE.

Decompondo a pseudo-class do link

<A> já sabemos para que serve(para criar um link). E como pode ver, usamos a sua tag para fazer uma personalização dentro da tag <STYLE> mas o cenário melhor mesmo é ter um documento CSS para isso. Mas aqui é só um exemplo então está tudo bem.

<:VISITED> como diz o próprio nome, quando a gente visita um link, ele por padrão muda de cor, né? Aqui a gente mudou a cor que fica após você clicar em um link.

<:ACTIVE> como também já diz o próprio nome, no instante que você clica em um link, ele muda de cor, como fizemos nesse exemplo aqui. Mas podemos fazer muito mais.

:HOVER a essa altura nós já sabemos para que serve seu propósito mas só para não deixar batido: Há um comportamento quando você passa o mouse por cima do texto ou do conteúdo.

Pseudo-Elementos

Pseudos-Elementos são como efeitos especiais que você adiciona com CSS em partes específicas de um elemento(Uma tag P, A, h2, etc por exemplo) sem precisar mudar nada no código HMLT.

Tipos de pseudo-elementos:

  • ::before
      • Use ::before para colocar algo no começo do texto, bloco ou alguma outra tag
  • ::after
      • Use ::after no final do conteúdo
  • ::first-line
      • Use ::first-line para mudar somente a primeira linha do texto
  • ::first-letter
      • Use first-letter para mudar somente a primeira letra (tipo deixar a primeira letra gigante como nos jornais)

Veja como ficou ao colocar o pseudo-element nos link externos:

        
            .a1::before{
                content: '♬';
            }

            .a1::after{
                content: '♬';
            }

Os pseudo-elementos começam com <::> e ajudam a deixar o site mais bonito e arrumado.
